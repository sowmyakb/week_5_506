The easiest way to plan your meals and automatically scale recipes by servings — for people who want a quick weekly meal plan with adjustable portions, without doing manual calculations.

Project brief

8 sections
1 Project name and pitch:
🍽️ Foodie — a Recipe Scaler and Meal Planner web app. Users search recipes via the Edamam API, automatically scale ingredient quantities by serving size, and assign recipes to days in a simple weekly meal plan.
2 Team members and roles:
Sam — server-side development, backend logic, API integration
Asia — frontend, UI/UX, JavaScript & CSS
Justin — database management, security, system design
Sowmya Korasikha — coordinator, CONTRACTS.md owner, integration testing, PR review
3 Project type:
Recipe Scaler & Meal Planner — one of the seven project ideas from the Week 4 lecture's "Example Project Ideas" slide. Per the slide: save recipes, scale servings, build weekly grocery lists, powered by a Nutrition API.
4 Endpoints planned:
GET /recipes/search — search recipes via Edamam API
GET /recipes/<id> — fetch recipe details + nutrition
POST /recipes/scale — scale ingredient quantities
POST /mealplan — add recipe to weekly planner
GET /mealplan — retrieve weekly meal plan
DELETE /mealplan/<day> — remove meal from day
GET /nutrition/<id> — nutrition breakdown per recipe
5 Web-based data sources:
Primary API: Edamam Recipe Search API
Specific endpoint: GET https://api.edamam.com/api/recipes/v2?type=public&q=<query>&app_id=<id>&app_key=<key>
Free-tier limits: approximately 10 monthly active users; limited requests per user per day; no caching or storing recipe data — suitable for a demo/portfolio project rather than large-scale production.
Backup API: USDA FoodData Central API — free, ~1,000 requests/hour per IP, but only raw food/nutrition data (no recipes or images).
6 Schema sketch:
Users(id, username, password_hash, created_at)
Recipes(id, api_id, name, image_url, calories, protein, carbs, fat)
Ingredients(id, recipe_id, name, quantity, unit)
MealPlans(id, user_id, day_of_week, recipe_id, servings)
